## Description

Sample small application to demonstrate usage of the TypeORM Entity Subscribers

## Installation

```bash
$ npm install
```

### Requirements

- **PostgreSQL** database with a database named as: **shop**

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
